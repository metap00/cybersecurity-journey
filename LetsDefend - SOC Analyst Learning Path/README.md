## Alert Investigations
Writeups from the SOC simulator — real alerts, real analysis.
Each writeup documents: what the alert was, how I investigated,
what I decided, and why.

- 

## Skills Being Built
- Alert triage and severity classification
- Distinguishing true threats from false positives
- IOC identification in log data
- Following playbooks and SOPs
- Building investigation timelines
- SIEM log analysis and event correlation

## Why This Matters
This is the closest thing to real SOC experience without
being in a SOC. Every module and investigation here maps
directly to what a Junior SOC Analyst does on day one.
